# こうかとんラン

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 横スクロールでジャンプして障害物をさけるゲーム
* 参考URL：[サイトタイトル](https://dinorunner.com/jp/)

## ゲームの遊び方
## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画

### 分担追加機能
* スターを作成する　(担当: けんしん)
* しゃがみ,障害物 (担当: ほりえ)
* 残機,スピードアップ微調整 (担当: りお)
* 未定(担当:　やぎりく)
* 未定(担当: かんた)
